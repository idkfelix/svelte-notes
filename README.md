# Svelte Notes
sveltekit app to view markdown notes from a git repo

## Run Locally
1. Clone git repo
```
$ git clone https://github.com/idkfelix/svelte-notes.git
```
2. Install npm packages
```
$ npm install
```
3. Create and configure `.env` file
```
# example: idkfelix/notes
USER_REPO=
# example: ghp...
KEY=
```
4. Run the app
```
$ npm run dev
```
